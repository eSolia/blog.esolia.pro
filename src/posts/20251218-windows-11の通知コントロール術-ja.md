パソコン作業中に画面の端から突然ピョコンと現れる通知。 
メールやTeamsメッセージ、Windows Updateの案内など、便利な半面「集中力を削ぐ」原因にもなります。 
しかし、やみくもに通知をオフにすると 重要なアラートや業務連絡を見逃すリスクも…
今回は Windows 11の通知設定を上手にコントロールする方法 と、設定時の「落とし穴」をご紹介します。

## 通知をオフにする方法

#### アプリ全体の通知をオフにする
1. 設定 → システム → 通知 
2. 「通知」のスイッチをオフ 
→ すべてのアプリからの通知が表示されなくなります

#### アプリ別にオフにする
同じ画面の「アプリやその他の送信者からの通知」から不要なアプリを選び、通知スイッチをオフ 

#### ロック画面での通知非表示
1. 設定 → システム → 通知 
2. 「ロック画面に通知を表示する」をオフ 
→ 外出先や共有PCでの覗き見防止に有効

## 「フォーカス」で集中しながら通知管理
Windows 11 では「集中モード」の代わりに **フォーカス** が搭載されています。 
フォーカスは、指定した時間だけ通知やバッジ表示を抑え、作業に集中できるモードです。

#### フォーカスの使い方
1. 設定 → システム → **フォーカス** 
2. 「フォーカスタイマー」で時間を設定（例：30分） 
3. フォーカス中は以下が自動的に有効になります
   * 通知の抑制（優先通知のみ許可） 
   * タスクバーアイコンのバッジ非表示
   * TeamsやOutlookの通知を一時停止

## 知らないと困る落とし穴
* **重要通知も抑制される可能性** 
→ フォーカス中でも受けたいアプリは「**優先通知**」に登録
  （設定 → システム → 有線通知から確認可能） 
* **Windows Update の再起動警告を見逃す** 
→ 長時間フォーカスを使う場合は事前に更新確認を 
* **アプリ側通知設定と連動しない** 
→ アプリ内の通知設定がオフだと、Windows設定をオンにしても出ない 
* **フォーカス終了後の通知が一気に届く** 
→ 大量の通知で見落としやすいので、終了後は通知センターを必ず確認

## ビジネス向けおすすめ設定例
**■業務時間（例：9:00〜17:00）**
* 通知オン（Teams・Outlook・セキュリティのみ許可） 
* SNSやニュースアプリはオフ
  
**■集中作業タイム** 
* フォーカス30〜60分＋優先通知リストに重要アプリを登録 

**■会議中** 
* Teams通知のみ許可、それ以外はフォーカスで遮断

## まとめ 
Windows 11 の**フォーカス**を使えば、通知を完全に切らずに集中力と情報キャッチの両立 が可能です。 
> [!TIP]
> ・通知設定とフォーカスを使い分ける 
> ・優先通知リストで重要な情報だけ受け取る 
> ・フォーカス終了後は通知センターを確認

正しく設定して、仕事効率を最大化しましょう。
