私たちが毎日何気なく使っているキーボード。普段は当たり前のように文字を入力していますが、「なぜこんな並びになっているの？」、「F1～F12はなんのためにあるの？」、「Fnキーって何？」と疑問に思うことはありませんか。今回は、キーボードの配列がどのようにして現在の形になったのか、そして、ファンクションキーとFnキーの役割について探っていきましょう。 

<!--more-->

## QWERTY配列 
現在、世界で最も普及しているキーボード配列は「QWERTY」配列です。その名称は、アルファベットの最上段の左から6文字の並び「Q-W-E-R-T-Y」に由来しています。 
この配列の起源は1870年代、クリストファー・ショールズらが初期のタイプライターを開発した時期にさかのぼります。当時の機械式タイプライターは、隣接するキーを続けて高速で打つと、タイプバー（文字を打刻する金属のアーム）が互いに干渉し、絡まってしまうという技術的な課題を抱えていました。 
この問題を解決するための一つの説として、ショールズは頻繁に続けて打たれる文字の組み合わせ（英語では「TH」や「HE」など）のタイプバーをできるだけ離して配置したと言われています。これにより、打字速度がある程度抑制され、タイプバーの衝突や絡まり（ジャム）を防ぐ効果があったと考えられています。 
このように、当時の技術的制約への対策として生まれた配列が、150年近く経ったデジタル時代の今日でも、社会的な慣性（デファクトスタンダード）によって世界標準として使われ続けているのです。 

[※参照URL：QWERTY配列](https://ja.wikipedia.org/wiki/QWERTY%E9%85%8D%E5%88%97){target="_blank" rel="noopener"}

## DVORAK配列 
1930年代、オーガスト・ドヴォラック博士は、QWERTY配列の非効率性に着目し、より合理的なキー配列の開発に取り組みました。DVORAK配列は、英文の文字出現頻度や人間工学に配慮して設計された代替え配列とされています。 

DVORAK配列の主な特徴は以下の通りです。 
* 母音を左手のホームポジションに集中配置、最も頻繁に使用する子音を右手のホームポジションに配置することで、打字のリズムを改善。 
* 上下の段への指の移動距離を最小化し、疲労を軽減。 
* 両手を交互に使用する単語の割合を増やし、打字速度の向上を図った。
  
ドヴォラック博士自身の研究によれば、DVORAK配列はQWERTY配列に比べて、タイピング効率の向上や、疲労度の軽減が期待できるとされました。しかしながら、既存のQWERTY配列の支配的な市場シェアと、ユーザーの慣れによって、DVORAK配列は広く普及することはありませんでした。現在では、主要なOSで標準サポートされており、効率性を重視する一部のユーザーやプログラマーを中心に利用され続けています。 

[※参照URL：Dvorak配列](https://ja.wikipedia.org/wiki/Dvorak%E9%85%8D%E5%88%97){target="_blank" rel="noopener"}

## JIS配列 
日本で広く普及しているキーボードは「JIS配列」です。これは日本産業規格（JIS）で規定された配置で、日本語入力の効率化を目的に設計されています。

その主な特徴は以下の通りです。 
* 日本語入力のための専用キー: 「半角/全角」キー（入力モード切り替え）、「変換」「無変換」キー（漢字変換の操作）、「カタカナ・ひらがな」キーが独立して配置されています。 
* 独特なEnterキーの形状: 多くの場合、縦長の大きなEnterキーを採用しており、遠い位置からでも打ちやすくなっています。 
* 記号の配置の違い: US配列と比べて、記号の位置や種類（例えば「@」キーの位置）が異なります。

これらは、ローマ字入力やかな入力における「漢字変換」という日本語特有の入力プロセスを強力にサポートするための工夫です。 

一方、海外で主流の「US配列」は、これらの日本語専用キーを持たないため、キーの総数が少なく、シンプルで見た目がすっきりしています。記号の配置がプログラミング言語に適していることや、物理的なキー配置の統一感から、プログラマーや英語を多用するユーザー、またこだわりを持つPCユーザーに好まれる傾向があります。

## Fnキーの機能 
ノートパソコンやコンパクトキーボードには「Fnキー」（Functionキーの略）が搭載されています。これはキーボードの拡張修飾キーと呼ばれ、限られたキー数で更多的な機能を割り振るための、いわば「魔法のキー」です。 

デスクトップ用のフルサイズキーボードと違い、筐体が小さいノートPCなどでは物理的なキーの数に制約があります。Fnキーは、一つのキーに二つの機能を持たせることで、この制約を巧みに解決します。FnキーはShiftキーやCtrlキーのように、単独では機能せず、必ず他のキーと同時に押して（組み合わせて）使用します。 

代表的な機能の組み合わせは以下の通りです。 
* 画面の明るさ調整：<kbd>Fn</kbd> + <kbd>F5</kbd>（暗くする）/ <kbd>Fn</kbd> + <kbd>F6</kbd>（明るくする） 
* 音量調整：<kbd>Fn</kbd> + <kbd>F2</kbd>（音量を下げる） /<kbd>Fn</kbd> + <kbd>F3</kbd>（音量を上げる） / <kbd>Fn</kbd> + <kbd>F1</kbd>（ミュート切り替え） 
* 表示出力の切替： <kbd>Fn</kbd> + <kbd>F7</kbd>（PC本体・外部ディスプレイ・複製などのモード切替） / <kbd>Fn</kbd> + <kbd>F8</kbd>（プロジェクタ等への出力） 
* その他便利機能： <kbd>Fn</kbd> + <kbd>スペース</kbd>（キーボードバックライトの点灯/消灯） / <kbd>Fn</kbd> + <kbd>F9</kbd>（タッチパッドの有効/無効切り替え） / <kbd>Fn</kbd> + <kbd>Esc</kbd>（Fnキーをロックする機能。F1～F12キーの本来の機能と、拡張機能の優先度を入れ替えられる機種あり） 
※メーカーや機種により割り当ては異なりますので、ご注意ください。 

このように、Fnキーの組み合わせを使いこなせば、面倒な設定画面を開くことなく、基本的なハードウェア機能を素早く操作することができます。 

## おわりに 
パソコンのキーボードは毎日触れるものだからこそ、「なぜこうなっているのか」を知っておくとより便利に使えます。何気なく押しているあのキー、よく目にするのに使ったことのないあの機能には、歴史的な経緯や、作業効率を高めるヒントが隠されているかもしれません。これを機会に、ご自身のキーボードのファンクションキーの機能を確認してみてはいかがでしょうか。
