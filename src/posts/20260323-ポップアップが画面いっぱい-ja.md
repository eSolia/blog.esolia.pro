ファイルを保存しようとしたときに、保存先を選ぶポップアップ画面が突然最大化されてしまい、サイズを自由に変更できず困った経験はありませんか？
このブログでは、「名前を付けて保存」画面のサイズ調整方法と、もし最大化された状態で表示された場合の画面サイズの戻し方についてご紹介します。

<!--more-->

## ポップアップ画面とは
そもそもポップアップ画面とは、普段の作業画面とは別に、必要なときだけ表示される小さなウィンドウのことを指します。ファイルを保存するときに表示されるのは、Windowsの標準機能である「名前を付けて保存」ダイアログです。この画面では、保存先のフォルダーを選択したり、ファイル名を入力したりします。
そのため、画面が大きすぎたり小さすぎたりすると操作しづらく感じることがあります。

## ウィンドウサイズ調整は可能か？
保存ダイアログはOS標準の仕組みを利用して表示されているため、自由なサイズ変更が許可されていない場合があり、特に古いOSや古い業務アプリケーションで固定サイズになっているケースが多いようです。
しかし、近年のWindows 10以降では、保存ダイアログの改良が進み、結論としてほとんどの場合ウィンドウサイズの調整が可能となっています。
保存画面がエクスプローラー形式（Windowsのエクスプローラーでファイルやフォルダーを表示する際の形式のこと）で表示される場合、ウィンドウの四隅または右下隅をドラッグすることで自由に拡大・縮小が可能です。

{{- comp.icon({ name: "cursor-click", size: 4, color: "blue" }) -}}操作方法
1. 保存ポップアップを開く。
2. 画面の右端または右下にマウスカーソルを移動。
3. カーソルが「↔」や「↕」に変わるかを確認。
4. 変わった場合、ドラッグしてサイズを調整。
※アプリによってはこの操作が無効化されている場合がありますので、ご注意ください。

## 画面いっぱいに広がったウィンドウのサイズ調整は？
では、画面いっぱいに広がってしまった「名前を付けて保存」ダイアログボックスを、元のサイズに戻す方法をご紹介します。一度下記のスクリーンショットのような状態になると、次回以降も同じ大きさで表示され続けてしまいます。筆者も何度もこの現象に悩まされてきました。
今回は、この現象を解決する3つの方法を順番にご説明します。

### 方法１： タイトルバーをダブルクリック

1. ダイアログボックスが最大化されている状態では、右下隅にカーソルを合わせてドラッグしてもサイズを変更できません。
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing desktop files" src="/uploads/202508a-file-saving-popup-ja1.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>

2. そのような場合は、「名前を付けて保存」ダイアログボックスの上部にあるタイトルバーをダブルクリックします。
3. これでダイアログボックスは元のサイズに戻ります。
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing desktop files" src="/uploads/202508a-file-saving-popup-ja2.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>

## 方法２：タイトルバーのショートカットメニュー「元のサイズに戻す」
1. 最大化されたダイアログボックスのタイトルバーを右クリックすると、「元のサイズに戻す」オプションが表示されます。
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing desktop files" src="/uploads/202508a-file-saving-popup-ja3.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>
2. 「元のサイズに戻す」を選択すると、ダイアログボックスが元の大きさに戻り、マウスカーソルでドラッグしてサイズ変更が可能になります。
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing the file name field" src="/uploads/202508a-file-saving-popup-ja4.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>

## 方法3：「フォルダーの非表示」からサイズ調整
1. 最大化されたダイアログボックスの画面左下にある「フォルダーの非表示」をクリックしてください。
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing the “Hide folders”option" src="/uploads/202508a-file-saving-popup-ja5.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>
2. ダイアログボックスの最大化が解除され、右下隅をドラッグしてウィンドウのサイズを調整できるようになります
<figure class="flex flex-col justify-start items-left">
  <img alt="Screenshot of a Windows“Save As”dialog box showing showing the file name and settings" src="/uploads/202508a-file-saving-popup-ja5.png" width="500px" transform-images="avif webp png jpeg 500@2">
</figure>
3. もしくは、上記ステップで「フォルダーの参照」をクリックすると、ダイアログボックスは一定サイズに戻り、最大化はされません。
<figure class="flex flex-col justify-start items-left">

</figure>

## おわりに
多くのアプリでは、「名前を付けて保存」ウィンドウのサイズや位置を手動で変更すると、次回も同じ状態で表示されることが一般的です。ただし、一部のアプリでは起動時にサイズや位置が初期化（リセット）される場合もあります。ほとんどの場合、保存画面のサイズは自分に合った大きさに調整できるため、少しの工夫で作業のストレスを大幅に軽減できます。保存画面のサイズ調整にお困りの際は、本篇の内容が皆様のお役に立てれば幸いです。
